## Technologies Used 👨🏽‍💻:
1. Nodejs & Express
2. Mongodb Atlas(Database)
3. HTML , CSS , JavaScript
4. EJS , Bootstrap


## Features 📋
🔥 User Authentication.\
🔥 Create new Blog.\
🔥 Delete Blog with one click.\
🔥 User Portfolio.\
🔥 Users can see others portfolio.
